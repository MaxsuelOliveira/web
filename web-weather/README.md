# Weather Widget

Widget de previsão do tempo em JavaScript puro com:

- busca por CEP usando ViaCEP
- geolocalização do navegador com fallback por localização aproximada
- previsão atual e próximas horas com Open-Meteo

## Como usar

Abra o projeto em um servidor estático e use:

- o botão Minha localização para carregar o clima da posição atual
- o campo de CEP para buscar a previsão por cidade

## APIs usadas

- ViaCEP
- Open-Meteo Forecast API
- BigDataCloud Reverse Geocode Client
